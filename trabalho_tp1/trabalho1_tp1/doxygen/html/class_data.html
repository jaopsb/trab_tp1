void Data::set_data ( string  valor)

Adiciona o valor da data a classe Data.

A data deve ser descrita da forma 'DD/MMM/AAAA', não será valido anos abaixo de 2000 e acima de 2099. Anos bissextos são considedrados, o mes devera ser uma das siglas validas: jan,fev,mar,abr,mai,jun,jul,ago,set,out,nov,dez.
